## :information_source: How To Use

### Prerequisites
- Docker
### Usage
From your command line:

```sh
# Clone repository
git clone https://github.com/mrlucasrib/PaymentControl.git
cd PaymentControl
# Install Docker if you do not have
docker-compose up -d
```
